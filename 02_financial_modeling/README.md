# Integrated Financial Modeling: DCF & LBO

## Overview
This project contains a fully integrated 3-statement model with DCF valuation and LBO analysis, plus workflow automation hooks.

## Contents
- `DCF_LBO_Model.xlsx` — Core Excel model
- `model_automation.py` — Python helper script for lightweight model tasks
- `screenshots/` — Visual snapshots of the model outputs

## Features
- Linked income statement, balance sheet, and cash flow model
- Multi-stage DCF with WACC and terminal value assumptions
- LBO returns analysis (IRR and MOIC)
- Scenario handling for Base, Bull, and Bear cases

## Usage
1. Open `DCF_LBO_Model.xlsx`
2. Enable macros
3. Run scenario automation from the `Automation` sheet
